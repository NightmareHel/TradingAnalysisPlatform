Quick Start Guide — macOS

Before You Begin — One-Time Setup

First time only. Once setup is complete, you just double-click run.command to start the app every time. Setup takes about 5 minutes.
1
Install Python (if you haven't already)

Visit python.org/downloads and download the latest version. Open the downloaded file and follow the installer. This takes about 2 minutes.

2
Unlock the downloaded files (one time only)

macOS blocks files downloaded from the internet. To unlock them:

  • Open Terminal (press Cmd + Space, type Terminal, press Enter)
  • Type xattr -cr (with a space at the end), then drag the project folder from Finder into the Terminal window
  • Press Enter

This removes the security block from all files. You only need to do this once.

3
Double-click run.command

A Terminal window will open and set everything up automatically. This takes about 30-60 seconds on first launch. Your browser will open when it's ready.

4
Enter your API keys

Your browser will open automatically. Click Settings in the left sidebar and paste your 4 API keys. Then click Save API Keys.

After the first time, just double-click run.command to start the app. No Terminal commands needed again.

Running an Analysis

1
Double-click run.command

The app opens in your browser automatically. Keep the Terminal window open — closing it stops the app.

2
Go to Dashboard and type a ticker

Examples: AAPL, MSFT, NVDA for US stocks. For Indian stocks: RELIANCE.NS, INFY.NS.

3
Click Analyze and wait 10-20 seconds

The app collects data from 7 sources and generates AI predictions for 4 timeframes: today, 1 week, 1 month, and 1 year.

4
Export as PDF (optional)

Click Generate PDF Report at the bottom of the Dashboard to download a formatted report.

Stopping the App

Close the Terminal window that opened when you launched run.command. The browser tab will stop working, which is expected.

Troubleshooting

Issue Fix
Browser doesn't open automatically Manually go to localhost:8501
Port already in use error Close any other terminal windows running the app, then try again
"Groq rate limit reached" Wait 60 seconds and click Analyze again
Settings page says keys are missing Go to Settings, re-enter keys, click Save API Keys
App won't open — "unidentified developer" Right-click run.command → Open → Open (see Step 2 above)
